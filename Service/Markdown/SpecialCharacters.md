# 特殊文字
* Markdown そのものの機能ではなく、GitHub の Web ページ上で見るだけなら、HTML の仕様で使える文字です。


## 特需文字
| 文字     | 入力       | 備考 |
|:--------:|:-----------|:-----|
| &copy;   | `&copy;`   | コピーライト |
| &reg;    | `&reg;`    | 登録商標 |
| &euro;   | `&euro;`   | ユーロ |
| &trade;  | `&trade;`  | トレードマーク |
| &hearts; | `&hearts;` | ハート記号 |
